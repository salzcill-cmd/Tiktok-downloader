## 🔌 API Documentation

### Endpoint

```http
GET https://api.ryzumi.net/api/downloader/tiktok?url={TIKTOK_URL}
```

### Contoh Request

```bash
curl "https://api.ryzumi.net/api/downloader/tiktok?url=https://vt.tiktok.com/xxxxx/"
```

### Response Format

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "title": "Judul Video TikTok",
    "cover": "https://thumbnail-url.jpg",

    "author_avatar": "https://avatar-url.jpg",

    "comment_count": 7890,
```

### Status Code

| Code | Meaning | Action |
|:---:|:---|:---|
| `0` | ✅ Success | Data tersedia |

## 🐛 Troubleshooting

### ❌ `Failed to fetch` / CORS Error

**Penyebab:** Browser memblok request langsung ke API eksternal.

**Solusi:** ✅ Gunakan `api.php` sebagai proxy backend.

```javascript
// ❌ Jangan fetch langsung ke API
fetch("https://api.ryzumi.net/...")  // Error CORS!

// ✅ Fetch ke backend sendiri
fetch("api.php?url=" + encodeURIComponent(tiktokUrl))  // Aman!
```

---

### ❌ `Unexpected token '<'`

**Penyebab:** PHP tidak berjalan, file dianggap HTML biasa.

**Solusi:** Pastikan file ber-extension `.php`:

```bash
# ❌ Salah
index.html

# ✅ Benar
index.php
```

---

### ❌ API Tidak Merespon

**Cek langsung via browser:**

```
https://domainkamu.com/api.php?url=https://vt.tiktok.com/xxxxx/
```

| Hasil | Arti | Action |
|:---|:---|:---|
| JSON muncul | ✅ Backend aman | Cek frontend |
| Error/Blank | ❌ PHP/cURL bermasalah | Cek `phpinfo()` |
| 500 Error | ❌ Server error | Cek error log |

---

### ❌ Video Tidak Bisa Di-download

| Penyebab | Solusi |
|:---|:---|
| Link TikTok invalid | Pastikan link lengkap (`https://...`) |
| Video private | Tidak bisa di-download |
| API rate limit | Tunggu 1-2 menit, coba lagi |
| Region block | Gunakan VPN jika perlu |

## 🔒 Security Best Practices

<div align="center">

| ⚠️ Jangan | ✅ Lakukan |
|:---|:---|
| Spam request ke API | Batasi rate limit di `api.php` |
| Upload file executable | Hanya upload `.php`, `.html`, `.css`, `.js` |
| Gunakan HTTP tanpa SSL | Selalu aktifkan HTTPS |
| Expose API key | Simpan secret di server-side |
| Biarkan tanpa rate limit | Tambahkan IP-based throttling |
